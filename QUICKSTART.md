# 快速开始指南

## 快速启动

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看网站。

### 3. 添加第一个作品

1. 访问 [http://localhost:3000/admin](http://localhost:3000/admin)
2. 点击"添加作品"按钮
3. 填写项目信息并保存

## 主要功能

### 🎨 首页展示
- 响应式网格布局展示所有作品
- 按分类筛选项目
- 美观的渐变设计和动画效果

### 📝 项目详情
- 完整的项目信息展示
- 图片和视频预览（支持点击放大）
- 代码高亮显示（支持多语言）
- 项目文档展示

### ✏️ 管理功能
访问 `/admin` 页面进行作品管理：
- ✅ 添加新作品
- ✏️ 编辑现有作品
- 🗑️ 删除作品
- 📦 管理媒体文件（图片/视频）
- 💻 管理代码文件
- 📄 管理项目文档

## 数据存储

所有数据存储在 `data/portfolio.db` SQLite 数据库中，数据库会在首次运行时自动创建。

## 项目分类

支持以下分类：
- 计算机图形学
- 流体仿真
- 自动驾驶
- 游戏开发
- 人工智能
- 其他

## 技术栈

- **前端框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **数据库**: SQLite (better-sqlite3)
- **动画**: Framer Motion
- **代码高亮**: react-syntax-highlighter
- **图标**: Lucide React
